# Library Manager Frontend

## Description

Library management application.

## Installation

```bash
$ git clone git@github.com:AdrianFadiga/library-manager-frontend.git
$ cd library-manager-frontend
$ npm install
```
## Running the app

```bash
# development
$ npm run dev
```
